# notedocal 🌙

**Note** + **To-Do** + **Cal**endar — заметки, список дел и календарь в одном приложении,
с поддержкой **григорианского** и **исламского (хиджра)** календарей.

Кроссплатформенное веб-приложение: открывается в любом браузере (телефон, планшет,
компьютер), работает офлайн, данные хранятся прямо на устройстве. Можно «установить»
на телефон как обычное приложение (PWA).

## Возможности

- 📅 **Календарь** — сетка месяца с двойными датами (григорианская + хиджра), сегодняшний
  день и выбранный день подсвечены. На днях с задачами/заметками — цветные точки.
- ✓ **Задачи** — список дел с приоритетами, сроками и отметкой выполнения.
- 📝 **Заметки** — быстрые текстовые заметки с редактированием.
- 🗓 **День** — кликнув по дню в календаре, можно добавить задачи и заметки именно на эту дату.
- 🌗 Светлая и тёмная темы (автоматически по системным настройкам).
- 💾 Все данные — в `localStorage` браузера, без сервера и регистрации.

Хиджра-даты вычисляются встроенным в браузер механизмом `Intl` (календарь
`islamic-umalqura`, Умм аль-Кура) — без внешних библиотек.

## Технологии

Vite · React · TypeScript.

## Запуск

```bash
npm install      # установить зависимости (один раз)
npm run dev      # запустить в режиме разработки → http://localhost:5173
npm run build    # собрать продакшен-версию в папку dist/
npm run preview  # посмотреть собранную версию
```

## Структура

```
src/
  types.ts              # доменные типы (Task, Note)
  lib/
    dates.ts            # утилиты дат, хиджра, сетка месяца
    storage.ts          # хранение в localStorage
  components/
    CalendarView.tsx    # календарь + панель дня
    TasksView.tsx       # список задач
    NotesView.tsx       # заметки
  App.tsx               # навигация и общее состояние
```
